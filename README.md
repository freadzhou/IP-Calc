## ✨ 功能特性

### 📡 IPv4 计算器

| 功能模块                 | 描述                                                         |
| ------------------------ | ------------------------------------------------------------ |
| **网络和 IP 地址计算器** | 输入 IP 地址和 CIDR，自动计算子网掩码、网络地址、广播地址、可用地址范围 |
| **CIDR → 子网掩码**      | 通过掩码位数快速计算子网掩码和可用地址数                     |
| **主机数 → 子网掩码**    | 根据需要的主机数量反推最优子网掩码                           |
| **IP 进制转换器**        | IP 地址在十进制、二进制、十六进制之间互转                    |
| **子网掩码 → CIDR**      | 将点分十进制子网掩码转换为 CIDR 表示法                       |
| **反掩码计算器**         | 计算反掩码/通配符掩码（用于 ACL 配置）                       |

### 🌌 IPv6 计算器

| 功能模块              | 描述                            |
| --------------------- | ------------------------------- |
| **IPv6 ↔ 数字互转**   | IPv6 地址与大整数之间的相互转换 |
| **IPv6 压缩/扩展**    | IPv6 地址格式压缩与完整展开     |
| **IPv6 地址范围计算** | 根据 IPv6 前缀计算子网地址范围  |

### 🛠️ 网络工具集

| 工具                | 描述                                                         |
| ------------------- | ------------------------------------------------------------ |
| **MAC 地址工具**    | 支持冒号、点号、无分隔等多种格式互转，自动生成 EUI-64 接口标识符 |
| **端口号速查表**    | 内置 100+ 常用服务端口，包含安全检测端口（恶意软件常用端口标记） |
| **VLSM 子网划分器** | 可变长子网掩码计算，自动规划多子网地址分配                   |
| **网络带宽计算**    | 根据文件大小和带宽估算理论传输时间                           |
| **IP 归属判断**     | 快速判断某个 IP 地址是否属于指定网段                         |
| **CIDR 聚合**       | 路由汇总工具，将多个连续子网聚合成更大网段                   |

## 📖 使用指南

### IPv4 网络计算

1. 在 **IPv4 计算器** 选项卡中，输入 IP 地址的四个八位组
2. 输入 CIDR 掩码位数（0-32）
3. 点击 **计算** 按钮，即可获得完整的网络信息

**示例：**

```
IP: 192.168.1.100 / 24
结果:
  - 子网掩码: 255.255.255.0
  - 网络地址: 192.168.1.0
  - 广播地址: 192.168.1.255
  - 可用地址: 254 个 (192.168.1.1 - 192.168.1.254)
```

### VLSM 子网划分

1. 切换到 **网络工具** 选项卡
2. 找到 **VLSM 子网划分器** 模块
3. 输入网络地址和原始掩码
4. 输入各子网需要的主机数量
5. 点击 **开始划分**，系统自动按最优方案分配

**示例：**

```
网络: 192.168.1.0/24
需求: 50, 20, 10, 5 台主机
结果:
  - 子网1: 192.168.1.0/26 (62 可用地址)
  - 子网2: 192.168.1.64/27 (30 可用地址)
  - 子网3: 192.168.1.96/28 (14 可用地址)
  - 子网4: 192.168.1.112/29 (6 可用地址)
```

### 端口号查询

1. 在 **网络工具** 选项卡中找到 **端口号速查表**
2. 可通过下拉菜单按分类筛选：
   - 系统服务 (0-1023)
   - 用户应用 (1024-49151)
   - 🔒 安全检测（恶意软件常用端口）
3. 支持关键词搜索（端口号或服务名）
